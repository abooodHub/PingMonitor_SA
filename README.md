# PingMonitor SA.v1 — أداة مراقبة البنق

تطبيق ويندوز لقياس البنق والجيتار وفقد البيانات لخوادم متعددة بواجهة حديثة ودعم العربية والإنجليزية.

## التحميل
- أحدث إصدار جاهز: `https://github.com/abooodHub/PingMonitor_SA/releases/latest`
- ملف الحزمة (zip) يحتوي ملف التشغيل `exe`

## المتطلبات
- Windows 10 أو أحدث
- .NET Framework 4.7.2

## التشغيل
1. نزّل الحزمة من صفحة الإصدارات.
2. فك الضغط وشغّل الملف:
   - `PingMonitor SA.v1 — أداة مراقبة البنق.exe`
3. اختر مزوّد الخدمة من الشريط العلوي وتابع القيم في الجدول.

## المزايا
- قياس البنق بشكل دوري مع مؤشرات لونية
- حساب الجيتار (Jitter) وفقد البيانات (Loss)
- تبديل اللغة عربي/إنجليزي
- أعلام الدول وقائمة مزوّدين قابلة للتصفية
- فرز الأعمدة حسب القيم
